# MvcMovie

Este es un proyecto de ejemplo desarrollado con Blazor en .NET 8 que demuestra cómo crear una aplicación sencilla para la gestión de una base de datos de películas. Incluye funcionalidades como:

- Listado de películas
- Agregar nuevas películas
- Editar películas existentes
- Eliminar películas

## Requisitos

- .NET 8 SDK
- Visual Studio 2022

## Ejecución del proyecto

1. Clona este repositorio.
2. Abre la solución en Visual Studio 2022.
3. Restaura los paquetes NuGet si es necesario.
4. Ejecuta el proyecto.

## Tecnologías utilizadas

- Blazor (.NET 8)
- Entity Framework Core
